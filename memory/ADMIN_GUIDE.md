# Kirti's Portfolio - Admin Guide

Welcome to your website admin panel! This guide will help you manage your portfolio completely on your own.

## Accessing the Admin Panel

**URL:** https://[your-domain].com/admin  
**Username:** admin  
**Password:** Kirti2024!

⚠️ **Important:** Change your password after first login!

---

## What You Can Do

### 1. **Home Page Management**
- Upload hero portrait image
- Edit tagline
- Update introduction text

### 2. **About Page Management**
- Upload portrait photo
- Edit background story
- Update approach philosophy
- Modify influences section
- Add/remove skills
- Change inspiring quote

### 3. **Projects Management**
- **Add New Project:** Click "Add Project" button
- **Upload Images:** Project photos automatically optimized
- **Add Media:** 
  - For music: Paste SoundCloud/audio URL
  - For videos: Paste YouTube URL
- **Edit Project:** Click edit icon on any project
- **Delete Project:** Click trash icon (confirms before deleting)

### 4. **Gallery Management**
- **Upload Images:** Drag & drop or click to upload
- **Categories:**
  - Portraits: Professional headshots
  - Moments: Candid, artistic shots
  - Work: Performance stills
  - Behind: Behind-the-scenes photos
- **Add Captions:** Describe each image
- **Delete Images:** Hover over image, click trash icon

### 5. **Events Calendar**
- **Add Event:** Click "Add Event"
- **Required Info:**
  - Event title
  - Date
  - Venue name
  - Location
- **Optional:**
  - Time
  - Description
  - Ticket URL (for BookMyShow, etc.)
- **Event Types:** Performance, Concert, Theatre, Workshop, etc.

### 6. **Contact Information**
- Update email address
- Update Instagram URL

### 7. **Messages Inbox**
- View all contact form submissions
- Mark messages as read
- Email addresses are clickable for quick reply

---

## Quick Tips

✅ **Image Sizes:**
- Hero/Portrait: 1200×1400px or larger
- Project Images: 1200×800px
- Gallery Images: 800×800px minimum

✅ **Best Practices:**
- Use high-quality images
- Keep descriptions concise and engaging
- Update events calendar regularly
- Check messages inbox weekly

✅ **Mobile Friendly:**
- All admin features work on tablet/phone
- Upload images from your phone's gallery
